These are select web app (Ruby on Rails) files from the project I did with Medeo.

application.css and graphs.css contain some minimal css work.
application_controller.rb is the base controller for the web app and provides database access to the other controllers that inherit from it.
_row.html.erb (Embedded RuBy) populates the rows of a an html table with the rows of data from dataset returned from the database
graphs_controller.sample.rb creates a variety of png graphs from a dataset

Languages used: Ruby on Rails, SQL, HTML, CSS
